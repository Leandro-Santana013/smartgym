# 🏋️ SmartGym - Plataforma de Gerenciamento de Treinos

Uma plataforma web inspirada no ecossistema de academias como a Smart Fit. O projeto foi desenvolvido para que os alunos possam gerenciar e acompanhar sua rotina de musculação de forma simples, digital e intuitiva diretamente pelo celular ou computador.

## 🚀 Funcionalidades Clave

* **Autenticação:** Tela de login e cadastro de usuários.
* **Controle de Treinos:** Seção dedicada para listar e selecionar os aparelhos da academia.
* **Personalização de Séries:** Configuração e visualização de:
  * Número de séries.
  * Repetições por exercício.
  * Carga/peso utilizado.
* **Interface Responsiva:** Otimizado para visualização em dispositivos móveis, facilitando o uso durante o treino.

## 🛠️ Tecnologias Cogitadas / Utilizadas

* **Frontend:** HTML5, CSS3, JavaScript / React
* **Backend:** Node.js / NestJS
* **Banco de Dados:** PostgreSQL / MySQL / MongoDB
