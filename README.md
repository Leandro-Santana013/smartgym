# 🍰 Amo Doce

Site de confeitaria artesanal desenvolvido como projeto prático da disciplina de **Programação de Sítios para Internet**.

## 📋 Sobre o projeto

O **Amo Doce** é um site para uma confeitaria artesanal que oferece bolos, cupcakes e doces feitos especialmente para cada ocasião. O projeto foi desenvolvido com foco em boas práticas de HTML, CSS e JavaScript, incluindo manipulação de DOM, eventos e versionamento com Git.

## 🚀 Como executar

Clone o repositório e abra o arquivo `index.html` no navegador:

```bash
git clone https://github.com/Leandro-Santana013/amo-doce.git
cd amo-doce
```

Não é necessário servidor — basta abrir o `index.html` diretamente no navegador.

## ⚙️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (ES6+)
- Git / GitHub

## ✅ Requisitos atendidos

- [x] Mínimo de seis páginas
- [x] Pelo menos quatro páginas com interação via eventos e DOM
- [x] Pelo menos uma página com manipulação completa do DOM (inserção, alteração, remoção e leitura)
- [x] Pelo menos uma página com traversal/traversing no DOM
- [x] Código organizado em arquivos HTML, CSS e JavaScript separados
